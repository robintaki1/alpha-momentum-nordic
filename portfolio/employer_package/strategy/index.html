How it works

What happens each month

The rule set is intentionally simple. The explanation below keeps the jargon out of the way.

1. Measure Check who has been strongest recently

For every eligible stock, compare the price from 12 months ago with the price from 1 month ago. That captures the trend while skipping the latest month.

2. Sort Put the strongest stocks first

Only the eligible names are ranked, after tradability, capacity, and data checks have passed.

3. Hold Own the top 8 equally

The selected names each get the same weight, so no single stock dominates the portfolio.

4. Sell Remove anything that no longer qualifies

At the next rebalance, anything no longer in the top 8 is sold. If the market filter is off, the strategy can stay in cash.

Plain-English takeaway: each month the portfolio looks for recent winners, keeps the best eight, and steps aside when the broad market looks weak.
