Monitor rules

What happens here

  • Do Watch realized months, overlap, frictions, and drift in the frozen books.
  • Do not Do not reopen the search or retune parameters because of short-term noise.
  • Cadence One trade batch per month: signal month, next execution date, then the new holding month.
  • Review clock Earliest review is August 2026; preferred review is February 2027.
  • Source Copied into the portfolio so this route stays self-contained.

How it operates

This is a point-in-time replay, not a live broker feed. The page makes the data source, execution date, friction model, and look-ahead guardrails explicit.

Data provenance

Historical daily parquet snapshots in `data/`, replayed at the monthly check.
