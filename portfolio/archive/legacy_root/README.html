This folder is the shareable entry point for the project package. The main assets live in:

  • `alpha_momentum_validation_package/dashboard.html`: visual landing page for the whole package
  • `alpha_momentum_validation_package/cadence_compare_summary/dashboard.html`: selected candidate dashboard
  • `alpha_momentum_validation_package/research_audit_dossier.html`: written summary of the current repo truth
  • `alpha_momentum_validation_package/cadence_compare_summary/cadence_comparison_report.html`: written package verdict
  • `alpha_momentum_validation_package/PROJECT_STATUS.html`: one-page narrative status summary

If you want the simplest route through the visuals, start with `alpha_momentum_validation_package/dashboard.html`.
